소비기한 등록 방법

등록/수정 화면의 소비기한 입력란에 적용됩니다. 언제든 다시 바꿀 수 있습니다.

로그인 없이도 모든 기능을 그대로 사용할 수 있습니다. 기기 변경이나 재설치 후에도 데이터를 유지하려면 백업을 켜주세요.

백업을 켜면 사진을 포함한 화장품 목록과 신호등 기준이 Google 계정에 안전하게 저장되어, 앱을 삭제 후 다시 설치하거나 다른 기기에서 로그인해도 그대로 복원됩니다. Gemini API Key는 백업 대상이 아니며 항상 이 기기에만 저장됩니다.
