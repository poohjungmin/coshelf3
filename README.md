# Coshelf

> AI-powered cosmetic expiration date manager

사진 한 장으로 브랜드와 제품명을 인식하고,
화장품 소비기한을 손쉽게 관리할 수 있는 모바일 웹앱입니다.

🌐 **Live Demo**

https://poohjungmin.github.io/coshelf3/

---

## Features

- 📷 AI 기반 화장품 정보 인식 (Gemini Vision API)
- 📝 브랜드 / 제품명 / 카테고리 자동 입력
- 📅 소비기한 관리
- 🚦 D-day 신호등 표시
- ⚙️ 신호등 기준 사용자 설정
- ⭐ 즐겨찾기
- 🔍 검색
- 💾 데이터 백업 / 복원
- 📱 PWA 지원 (홈 화면 추가)

## Why Coshelf?

기존에는 메모나 캘린더로 소비기한을 관리했지만
등록 과정이 번거롭고 지속적으로 사용하기 어려웠습니다.

Coshelf는 사진 한 장으로 제품 정보를 인식하고,
소비기한을 간편하게 관리할 수 있도록 기획한 프로젝트입니다.

AI가 반환한 정보를 그대로 저장하지 않고,
사용자가 직접 확인·수정할 수 있도록 설계하여
신뢰성과 사용성을 높였습니다.
