# Coshelf

> 사진 한 장으로 등록하고, 화장품을 끝까지 사용하세요.

Coshelf는 화장품의 소비기한을 쉽고 편하게 관리할 수 있는 PWA(Web App)입니다.

기존의 메모나 캘린더 방식은 제품을 하나씩 직접 입력해야 하는 번거로움이 있었고, 결국 기록을 하지 않거나 소비기한을 놓치는 경우가 많았습니다.

Coshelf는 제품 사진만 촬영하면 AI가 제품 정보를 인식하여 등록을 도와주고, 소비기한이 다가오면 알림을 보내 사용자가 화장품을 끝까지 사용할 수 있도록 돕습니다.

# 🚀 Why Coshelf?

올리브영 세일이나 행사 기간에 화장품을 여러 개 구매하지만,
사용 중인 제품이 많아 소비기한을 놓치고 버리는 경우가 자주 있었습니다.

기존의 메모 앱이나 캘린더는 등록 과정이 번거로워 꾸준히 사용하기 어려웠고,
이 문제를 해결하기 위해 사진 한 장만으로 제품을 등록하고 소비기한을 관리할 수 있는 서비스를 직접 제작했습니다.
